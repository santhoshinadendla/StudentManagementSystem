# Student Management System
## Project Overview
The Student Management System is a Java Full Stack Web application developed to manage Student records efficiently.It performes CRUD (Create,Read,Update,Delete)operations using Java Servlets,JSP,JDBC,MYSQL,and Apache Tomcat.

## Features
User Login
Add Student 
View Student Information
Update Student Record
MYSQL Database Connectivity
Simple and User-Friendly Interface

## Technologies Used
Java
JSP
Servlets
JDBC
MYSQL
Apache Tomcat
HTML
CSS
Maven
Eclipse IDE

## Database
Database: MYSQL
Table :students
Fields:
          -id 
          -name 
          -age
          -course

## How to Run
Import the project into Eclipse
Configure Apache Tomcat
Configure MYSQL database
Update database credentials in DBConnection.java
Run the Project on Tomcat Server
Open the application in the browser
